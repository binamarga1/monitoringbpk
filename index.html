Inisialisasi Data ke Firestore
Hanya gunakan sekali saat pertama kali setup. Jika Firestore sudah berisi data, operasi ini dibatalkan otomatis dan data tidak akan tertimpa.

Tombol ini akan memuat 80 paket data awal (termasuk semua data lunas) ke Firestore. Setelah ini semua perangkat yang membuka aplikasi akan melihat data yang sama secara real-time.

Config Firebase Aktif
Memuat config...
Untuk mengisi atau mengganti config Firebase, buka file HTML ini dengan teks editor (Notepad++, VS Code, dll), cari baris apiKey: "GANTI_API_KEY" di bagian <script type="module"> paling atas, lalu ganti keenam nilainya dengan config dari Firebase Console. Jangan edit bagian <pre> atau teks lain di halaman ini — itu hanya tampilan panduan, bukan tempat mengisi config.
